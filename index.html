Introdução HTML

Elementos do head

São elementos que conversam com o navegador, como o título da página, links para arquivos CSS, scripts e metadados.

Elementos do body

São os elementos que compõem o conteúdo visível da página, como textos, imagens, links e botões.

Ligações

As ligações são feitas utilizando a tag <a>, que permite criar links para outras páginas ou seções dentro da mesma página. Exemplo: 

<a href="https://www.exemplo.com">Clique aqui</a>
